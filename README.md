# Invoice Generator Website

This is a simple and clean **Invoice Generator Web App** built using **HTML, CSS, Tailwind CSS, JavaScript, and PHP**.  
The project allows users to create invoices, view reports, and manage basic settings through an easy-to-use interface.

---

## Features
- User login page  
- Dashboard with navigation  
- Create and view invoices  
- Download invoice/report as PDF  
- Settings page  
- Responsive UI with Tailwind CSS  
- Simple PHP backend

---

## Tech Stack
- **HTML**  
- **CSS / Tailwind CSS**  
- **JavaScript**  
- **PHP**  
- **MySQL (optional)**

---

## How to Run
1. Download or clone the repository.  
2. Move the project folder to your `htdocs` directory (for XAMPP).  
3. Start **Apache** (and MySQL if needed).
